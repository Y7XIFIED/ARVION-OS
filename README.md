# Arvion

> Futuristic browser-native desktop experience built with React, TypeScript, and motion-first UI.

[![Build](https://img.shields.io/badge/build-passing-22c55e?style=flat-square)](https://github.com/Y7XIFIED/ARVION-OS)
[![License: MIT](https://img.shields.io/badge/license-MIT-0ea5e9?style=flat-square)](./LICENSE)
[![Version](https://img.shields.io/badge/version-0.0.0-111827?style=flat-square)](./package.json)

## Features

- Desktop-style interface inside the browser
- Smooth, cinematic interactions with Framer Motion + GSAP
- 3D visual depth powered by Three.js
- Fast build and dev workflow via Vite

## Installation

```bash
git clone https://github.com/Y7XIFIED/ARVION-OS.git
cd ARVION-OS
npm install
```

## Usage

```bash
npm run dev
```

For production build:

```bash
npm run build
npm run preview
```
